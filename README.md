## MemoryManager

### Description
Memory manager for blocks of fixed size.

### Prerequisites
- `CMake` (minimum version 3.1) 
- `C++14 compiler` 

### Build and run 
```
git clone https://github.com/olga-vorobiova/MemoryManager.git
cd memoryManager
cmake .
make
./testMemoryManager
```
Tested on macOS, Linux
